# Javachord

## Maven

Add the Jitpack repository to your build file

```xml

<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

Release artifact

```xml

<dependency>
    <groupId>com.github.zrdj</groupId>
    <artifactId>javachord</artifactId>
    <version>3.8.0</version>
</dependency>
```
